## 21:08 — WaveSpeed Skill 功能补全
- **问题**: 有时候跑着跑着就中断回复，无任何提示
- **根因**: OpenAI 格式 API 请求未设置 `max_tokens`，各 provider 默认值差异大（有的低至 16/256 token），达到上限后 `finish_reason: "length"` 被静默截断
- **修复**:
  - `callChatCompletions` (3306行): body 加入 `max_tokens: _config.maxTokens || 4096`
  - `callAIWithTools` OpenAI 路径 (4023行): 同上
- **语法检查**: ✅ 通过
- **注意**: Anthropic 路径已有 `max_tokens: 8192`，不受影响
