## ClawPanel 聊天发送修复
- **问题**: 实时聊天发送消息报错 `invalid chat.send params: at root: unexpected property 'provider'`
- **根因**: `ws-client.js` chatSend 方法将 `provider` 和 `model` 作为 chat.send 顶层参数传入，API 不接受
- **修复**: 删除 `ws-client.js:916-917` 两行 + 清理 `chat.js:2088` 的 modelRef 传参
- **参考**: `scripts/dev-api.js:10269` 正确实现不含 provider/model

## 晴辰助手停止按钮修复
- **问题**: 输入框为空时点停止按钮没反应（UI 没变化）
- **根因**: `stopStreaming()` 只 abort 控制器 + 设 flag，但不更新 UI（图标、刷新定时器），完全靠 finally 块异步更新，abort 传播路径中断时 UI 就卡死
- **修复**: `stopStreaming()` 里立即做 UI 刷新 — stopStreamRefresh() + 图标还原 + textarea focus，不依赖 finally
- **旧逻辑保留**: 输入框非空时排队（Enter/点击按钮都是排队）
- **新逻辑**: `if (_isStreaming) stop` → 不管输入有什么都先停
