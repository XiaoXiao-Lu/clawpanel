## 修改
- `scripts/serve.js`
  - 为 `resolveStaticFilePath` 增加可注入 `distDir`，便于测试使用临时目录覆盖真实文件系统边界。
  - 为 `serveStatic` 增加可注入 `distDir` 选项，并导出 `serveStatic`，让测试可以启动内存 HTTP server 直接验证响应行为。
  - 保持生产默认行为不变，CLI 仍使用默认 `DIST_DIR`。
- `tests/serve-static-security.test.js`
  - 增加临时 dist 目录和本地 HTTP server 测试 helper。
  - 覆盖 `/assets/*.js` 的 immutable cache header。
  - 覆盖深链 SPA fallback 返回 `index.html` 且 HTML 禁缓存。
  - 覆盖缺失资源、encoded traversal、malformed encoding 的 HTTP 层 404/403/400 响应。
  - 使用 `http.request` 传入原始 path，避免 `fetch` 预先规范化 `%2e%2e` 导致无法测到服务器自身防护。
