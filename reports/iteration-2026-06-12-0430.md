## 本轮修复

### P2: 修复 light 主题 `--aether-sunken` 缺失

**文件**: [variables.css:485](src/style/variables.css#L485)

**问题**: light 主题 (`@media (prefers-color-scheme: light)`) 未覆盖 `--aether-sunken`，导致 `--surface-sunken` 在浅色模式下回退到 `var(--aether-sunken)` → `#0A0A14`（深色），背景对比度异常。

**修复**:
```css
@media (prefers-color-scheme: light) {
  :root:not([data-theme="dark"]) {
    /* 新增 */
    --aether-sunken:        #EEEEF3;
```

**效果**:
- `--surface-sunken` 在 light 主题下显示 `#EEEEF3`（`--light-bg-content`），与浅色背景协调
- 所有使用 `--surface-sunken` 的组件（memory、assistant、channels、notifications、services）在浅色主题下正确显示

### P3: Expert Teams showGroupTemplatePicker innerHTML → createElement

**文件**: [expert-teams.js:853](src/pages/expert-teams.js#L853)

**问题**: 模板选择器弹窗用 innerHTML 拼接动态内容，代码不一致且可维护性差。

**修复**:
1. 移除 `items` 模板字符串拼接
2. 用 `createElement` + `setAttribute` + `textContent` 构建 DOM
3. `strong` 仍保留 `innerHTML`（因为 `icon()` 返回 SVG 字符串，属于已知安全的 trusted HTML）

**效果**:
- 与 `showTagPicker` / `showMemberPicker` 等弹窗模式一致
- 消除动态 innerHTML 拼接的维护性风险
- 同步更新测试正则，支持 `setAttribute` 模式
