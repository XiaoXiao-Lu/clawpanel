## P2 建议优化（21项）

### CSS 设计系统
- 遗留别名 `--accent/--primary/--border` 仍被大量引用
- `--transition-fast` 与 `--ease-fast` 命名混用
- `--space-*` 双重命名系统
- Agent Office 专用颜色系统脱离设计令牌
- `.btn-icon` 在多个文件重复定义
- `about.css` 使用旧命名风格变量
- 缺少 `color-scheme` 声明

### JS 组件
- assistant.js 8980 行巨型文件需拆分
- modal.js trapFocus 监听器未清理
- security.js 密码强度判断不完整
- sidebar.js 搜索缺少 debounce
- chat.js renderAttachments 重复绑定事件

### 性能
- 缺少 CSS Containment 规则
- 过渡属性组合过多
- color-mix() 运行时计算
- grid 布局 auto-fill 性能
- 页面切换动画阻塞交互
- 伪元素创建额外合成层
- .chat-scroll-btn 缺少 GPU 加速
