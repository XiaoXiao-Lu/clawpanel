## Remaining Risks
- The lock only coordinates commands that use the project scripts. Direct `vite build` or manual edits to `dist` can still race the smoke.
- Stale-lock cleanup is age-based; a very long build over the stale threshold could be considered stale by another process, though the threshold is intentionally high.
- Playwright bundled Chromium is still missing in this environment, so `expert-teams:ui` continues to rely on system Chrome fallback.
