## 本轮修复

### P0: 添加缺失的 CSS RGB 变量别名

**文件**: [variables.css:413-415](src/style/variables.css#L413-L415)

**问题**:
多个文件使用 `rgba(var(--brand-green-rgb, ...))` 和 `rgba(var(--brand-amber-rgb, ...))` 语法，但这些变量未在 variables.css 中定义，导致 rgba 颜色失效，显示为透明。

**影响文件**:
- `src/lib/gateway-ownership.js` - 服务状态徽章
- `src/components/kernel-badge.js` - 内核状态徽章
- `src/pages/channels.js` - 通道配置提示

**修复**:
在 `variables.css` 的 RGB 别名区段添加：
```css
/* ─── Brand Color RGB Aliases (for rgba() usage) ─── */
--brand-green-rgb:  16, 227, 160;   /* success green */
--brand-amber-rgb:  255, 179, 71;   /* warning amber */
--brand-primary-rgb: 124, 111, 255; /* aether primary */
```

**效果**:
- 服务状态绿色徽章正确显示
- 警告提示橙色背景正确显示
- 内核状态徽章颜色正确显示
